# Form Rules

Defines form behavior in MedQuire.

---

## 1. Core Principles

### 1.1 Minimal Input
Only collect necessary data.

---

### 1.2 Clear Guidance
Users must understand:
- what to enter
- what went wrong

---

## 2. Form Types

---

### 2.1 Authentication Forms

Fields:
- email
- password
- confirm password

---

## 3. Input Rules

- use correct keyboard type
- label clearly
- use placeholders

---

## 4. Validation

- validate on submit
- show clear error messages

---

## 5. States

- default
- error
- loading
- success

---

## 6. Summary

Forms must:
- be simple
- be fast
- be clear
